Projet AlgoBox – Travail collaboratif
But du projet
Ce projet a pour objectif de concevoir et tester plusieurs algorithmes en AlgoBox.
Il s’inscrit dans un travail collaboratif réalisé à l’aide de GitHub, permettant à
plusieurs étudiants de travailler simultanément sur les fichiers .algo.

Les algorithmes développés permettent notamment de :
illustrer des structures conditionnelles,
utiliser des boucles,
manipuler des tableaux et des variables.

Organisation du projet
src/ : contient l’ensemble des fichiers AlgoBox
README.md : explication du projet et instructions d’exécution

Comment ouvrir les fichiers .algo:
Lancer le logiciel AlgoBox
Cliquer sur Fichier → Ouvrir
Sélectionner un fichier .algo situé dans le dossier src/
Le fichier principal du projet est main.algo

Comment exécuter et tester les algorithmes
Ouvrir un fichier .algo dans AlgoBox
Cliquer sur le bouton Exécuter
Suivre les instructions affichées (saisies clavier éventuelles)
Vérifier que les résultats obtenus correspondent aux attentes

Des tests ont été réalisés avec différentes valeurs d’entrée afin de valider le bon
fonctionnement des algorithmes.
